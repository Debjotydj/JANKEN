This project is a basic browser-based implementation of the well-known game called Rock Paper Scissors (also known as Janken). In this version, a user can play against the computer, and the computer's decision will be random.

This project includes a dynamic UI with buttons representing rock, paper, and scissors options, displaying results per round, as well as choices from the user and the computer. This project also contains a scoring system showing wins, losses, draws, and total points gained during each game.

One of the essential features of this project is the utilization of the browser local storage, allowing the score to persist even when refreshing the page. Furthermore, this project offers an option to reset the score.

For layout and design purposes, CSS is used to center the content and create a pleasant-looking interface. On the other hand, the main logic of the project, including randomness generation, DOM manipulation, and score persistence, is provided via JavaScript.

In conclusion, this project highlights essential web development features like DOM manipulation, event handling, and browser local storage.
